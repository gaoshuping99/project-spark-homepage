# Project Spark Homepage

This project is a high-quality landing page for **Project Spark**, designed to align with SAP's official branding and the style of the Professional Services Advisory Council 2025 event.

## Features
- **Modern SAP Aesthetic**: Utilizes SAP Blue (#0070b1), Deep Blue (#001e4d), and the Inter font family for a clean, professional look.
- **English Content**: Fully localized for an international audience.
- **Responsive Design**: Optimized for desktop, tablet, and mobile viewing.
- **Key Sections**:
  - **Overview**: Hero section introducing the mission.
  - **Partner Value**: Detailed value cards (Rebates, Lead Routing, Protection, etc.).
  - **Events**: Placeholder space for upcoming workshops and spotlights.
  - **Success Stories**: Peer case studies including the Accenture pilot.
  - **Registration**: Lead generation form for interested partners.
  - **Contact**: Direct contact information for project leads.

## Tech Stack
- **React**: Component-based architecture.
- **Tailwind CSS**: Utility-first styling for precise design.
- **Framer Motion**: Smooth animations for a premium feel.
- **Lucide React**: Minimalist iconography consistent with SAP's style.
- **Vite**: Ultra-fast development and build tool.

## How to Run
1. Navigate to the `website` directory:
   ```bash
   cd website
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Design Consistency
- **Logo**: Uses the official SAP logo provided in the project folder.
- **Color Palette**: Strictly follows SAP's corporate identity.
- **Typography**: Uses 'Inter' via Google Fonts as a high-quality substitute for SAP's proprietary '72' font, with '72' as a fallback.
